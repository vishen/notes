# Zig

## Download / Install

- https://ziglang.org/download/

```
$ curl  https://ziglang.org/download/0.6.0/zig-macos-x86_64-0.6.0.tar.xz -o zig-0.6.0
$ tar -xf zig-macos-x86_64-0.6.0.tar.xz
```

## Zig commands

### Running

- zig run helloworld.zig

### Building exe

- zig build-exe helloworld.zig
